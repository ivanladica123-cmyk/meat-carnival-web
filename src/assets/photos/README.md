# Fotografije / Photos

Drop owner-approved images into this folder and the site picks them up
automatically (jpg, jpeg, png, webp or avif). Name the file by its slot key:

| File name (key) | Where it appears | Suggested crop |
|---|---|---|
| `hero` | Homepage hero background (dark section) | wide, ≥1920×820 (21:9) |
| `kebab-lepinja` | Signature card 1 | 4:5 portrait |
| `cevapi-veliki` | Signature card 2 | 4:5 portrait |
| `hunters-burger` | Signature card 3 | 4:5 portrait |
| `carnival-plata` | Signature card 4 | 4:5 portrait |
| `terasa` | Terrace section | 4:3 landscape |
| `kebab` | Menu category tile | square-ish |
| `burgeri-junetina` | Menu category tile | square-ish |
| `burgeri-piletina` | Menu category tile | square-ish |
| `burgeri-divljac` | Menu category tile | square-ish |
| `grill` | Menu category tile | square-ish |
| `salate` | Menu category tile | square-ish |
| `prilozi` | Menu category tile | square-ish |
| `pica` | Menu category tile | square-ish |

Example: `hunters-burger.jpg` → replaces the "FOTOGRAFIJA USKORO" placeholder
on the Hunters burger card.

Photography direction (DESIGN_BLUEPRINT.md §5.3): hero item centered, black
matte plate or parchment, warm light, dark or warm-wood background. Prefer
WebP/AVIF ≤ 300 kB per image.
